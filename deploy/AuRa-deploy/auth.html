Авторизация

Войдите по Node ID и паролю. После входа откроется профиль с QR для синхронизации с приложением и VIP.

Админ: в Node ID введите root (если на сервере не задан другой AURA_ADMIN_LOGIN), в пароле — значение AURA_ADMIN_PASSWORD (по умолчанию в коде). Обычная нода: 8 hex, как в приложении.
